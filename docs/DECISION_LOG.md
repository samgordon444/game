# Decision Log

**Key milestone decisions for future reference**

---

## Fantasy Theme Chosen
**Decision:** Game set in fantasy realm (Harry Potter / Lord of the Rings inspired)
- Relatable magical being (wizard or elf) as player character
- Fantasy landscapes and magical elements

---

## Side-Scrolling Perspective
**Decision:** Game will be side-scrolling 2D
- Set routes/paths for exploration
- Horizontal movement focus
- Simpler to build and play

---

## Idle Mechanics Removed
**Decision:** Removed all idle/automatic wandering mechanics
- Player must actively play
- No automatic wandering or background progression
- No screensaver/idle mode

---

## Phaser 3 Chosen
**Decision:** Phaser 3.80.1 selected as game framework
- Vite 4.5.2 as build tool
- Web-first approach

---

## Art Style: Detailed Pixel Art
**Decision:** Detailed pixel art - beautiful and polished
- Emphasis on atmosphere and scenery
- Dynamic lighting and weather effects

---

## Simple Collection System
**Decision:** Simple item gathering system
- No complex menus or nested structures
- Simple inventory/collection display

---

*This log captures only key milestone decisions, not process details or status updates.*
